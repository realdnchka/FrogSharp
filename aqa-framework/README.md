## Framework for Automation QA

### Uses:
- Selenium
- RestSharp
- NUnit
- NLog
- .NET Framework 7.0

### Requirements:
- .NET Framework 7.0

### Features:
- Logger
- Api Helper
- Build-in CorePOM with all needed actions
- Build-in CoreTest with SetUp and TearDown
- Easy inheriting
- Flexible config

### Install:
``nuget install {name}``
